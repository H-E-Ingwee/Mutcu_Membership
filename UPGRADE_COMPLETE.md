#### API Endpoints (15+ New)

**Authentication:**
- ✅ `POST /api/auth/forgot-password` - Request password reset
- ✅ `POST /api/auth/reset-password` - Reset with token

**Events:**
- ✅ `GET /api/events/list` - List all upcoming events
- ✅ `POST /api/events/list` - Create event (admin)
- ✅ `POST /api/events/register` - Register for event
- ✅ `POST /api/events/register?action=unregister` - Cancel registration

**Announcements:**
- ✅ `GET /api/announcements/list` - List announcements
- ✅ `POST /api/announcements/list` - Post announcement (admin)

**Prayer:**
- ✅ `GET /api/prayer/list` - List prayer requests
- ✅ `POST /api/prayer/list` - Create prayer request
- ✅ `POST /api/prayer/list?action=intercede` - Add intercession
- ✅ `GET /api/prayer/list?action=answer` - Mark as answered

**Giving:**
- ✅ `POST /api/giving/donate` - Submit donation
- ✅ `GET /api/giving/donate?action=history` - Donation history
- ✅ `GET /api/giving/donate?action=summary` - Analytics (admin)

**Documents:**
- ✅ `GET /api/documents/list` - List documents
- ✅ `GET /api/documents/list?action=download` - Download document
- ✅ `POST /api/documents/list?action=upload` - Upload (admin)
